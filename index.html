Falando sobre tags

Voltar

Sobre tags aprendemos um pouco sobre como personalizar nosso texto para deixar ele em italico com a tag "i" e como deixar em negrito com a tag "strong" e para sublinhar um texto usamos a tag "u".

Aprendemos tambem nessa aula como criar um campo para digitar com a tag "imput type='text' ou 'number'ou 'color'".

E tambem aprendemos a colocar imagem com a tag "img src='endereço da imagem'" para mexer no tamanho da imagem antes do src colocamos width="numero" que a imagem irá diminuir.

Textos

Voltar

Para titulos e subtitulos vamos usar o "h" que ele vai de "h1" a "h6".

Para colocar paragrafos vamos usar a tag "p".

Para fazer um observação do texto usamos a tag "blockquote".

E para colocar uma linha entre os titulos e sub titulos usamos a tag "hr".

E para colocar um texto na parte de cima da linha como por exemplo o ao quadrado do 2 ao quadrado usamos a tag "sup".

Listas ordenadas e não ordenadas

Voltar

Para listas ordenas usamos a tag "ol". Só que para ela funcionar devemos colocar as filhas dela, colocamos a tag "li".

Para listas não ordenadas vamos usar a tag "ul", faremos a mesma coisa para ela funcionar, vamos colocar as filhas dela, que é a tag "li"

Links

Voltar

Para colocar links vamos usar a tag "a href"

Ao clicar no link para o usuario não sair de sua pagina usamos "target='_blank'" que será aberto outra aba do navegador.

Tambem podemos usar a tag "target='_self'" que irá substituir a aba do navegador atual.
